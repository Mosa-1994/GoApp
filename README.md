# OV Tijden

Real-time vertrektijden voor openbaar vervoer in De Meern en Vleuten.

## Features

- Bus vertrektijden vanaf halte De Meern, Veldhuizen (richting Utrecht en Vleuten)
- Trein vertrektijden vanaf Station Vleuten
- Live vertraging informatie
- Automatische refresh elke 15 seconden
- Werkt op mobiel als "app" via beginscherm

## Gebruik

Open de app via GitHub Pages:
**https://mosa-1994.github.io/GoApp/**

Of voeg toe aan je telefoon:
- **iPhone**: Open in Safari → Deel → "Zet op beginscherm"
- **Android**: Open in Chrome → Menu → "Toevoegen aan startscherm"

## Databronnen

Vertrektijden worden opgehaald via [drgl.nl](https://drgl.nl), die data ontvangt van NDOV/GTFS-RT.

## Lokaal draaien

Open `index.html` in een browser. Geen build process of dependencies nodig.
